# Controls
Up:    W or ↑ \
Left:  A or ← \
Down:  S or → \
Right: D or ↓ \
Shoot: Mouse 1 \

Pause: 	   Space \
Game Over: Backspace (once) \
Exit:      Backspace (twice)

# Cheats
Add Health: 	   +
Decrease Health: -
